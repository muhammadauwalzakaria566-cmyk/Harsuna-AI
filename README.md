# Harsuna AI

**Building foundational NLP infrastructure for the Hausa language.**

A project by [Khalifa TechBridge](https://github.com/muhammadauwalzakaria566-cmyk) — addressing the well-documented gap where mainstream AI (ChatGPT, Claude, Gemini) underperforms on Hausa and other African languages due to underrepresentation in training data.

---

## Why This Project Exists

Hausa is spoken by an estimated 60–80 million people, yet it remains massively underrepresented in the datasets that train today's AI systems. This leads to more mistakes, lost nuance, and missing cultural context for Hausa speakers — from basic tasks like text-to-speech to more complex language understanding.

Harsuna AI ("harsuna" = languages, in Hausa) is a long-term effort to help close that gap — starting small, starting honest, and building in public.
